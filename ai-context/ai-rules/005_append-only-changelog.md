# Правило 005: Любые file changes фиксируются в append-only changelog

## Статус

Действует.

## Правило

Если в рамках задачи были изменены, добавлены или удалены файлы проекта,
агент обязан добавить запись в `ai-context/changelog`.

## Обязательный подход

- один файл на день: `YYYY-MM-DD.md`;
- новые записи добавляются только в конец файла;
- старые записи не переписываются без прямого запроса пользователя;
- запись должна перечислять фактические изменения, а не абстрактные формулировки.

## Практический смысл

История работы AI должна жить внутри репозитория как прозрачный и понятный
операционный журнал.
