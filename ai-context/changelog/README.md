# Changelog

Эта директория хранит ежедневные append-only записи об изменениях, которые AI
вносил в проект.

## Формат

- один файл на день: `YYYY-MM-DD.md`;
- новые записи добавляются в конец файла;
- каждая запись должна содержать:
  - время;
  - короткий заголовок;
  - список измененных файлов или областей;
  - краткое описание фактически выполненных изменений.

## Пример

```md
## 14:30 - Обновлен baseline ai-context

- Файлы: `ai-context/tasks/task-rules.md`, `ai-context/README.md`
- Что сделано: уточнены execution-правила задач и политика обновления baseline.
```
