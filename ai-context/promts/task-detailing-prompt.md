# Промт: Детализирование задачи

## Назначение

Используй этот промт, когда пользователь просит: `детализируй 01-001`.

## Готовый промт

```text
Детализируй задачу `01-001` из `ai-context/tasks/task-list.md`.

1. Найди строку задачи с кодом `01-001`.
2. Создай файл `ai-context/tasks/task-details/01-001.md` по шаблону `ai-context/tasks/task-details/_template.md`.
3. Заполни обязательные разделы:
- короткое описание;
- цель;
- контекст;
- что сделать;
- критерии готовности;
- обязательные дополнительные условия;
- связанные правила;
- связанные файлы.
4. Сохрани смысл исходной формулировки задачи.
5. Если есть применимые project-specific правила из `ai-context/rules`, добавь ссылки на них.
6. Обнови статус задачи в `task-list.md` на `🟠`, если работа по ней еще не началась.
7. Создай отдельный changelog-файл в `ai-context/changelog` по правилу «одно изменение = один файл».

Не делай:
- изменение смысла исходной задачи без запроса пользователя;
- пропуск обновления статуса;
- пропуск отдельного changelog-файла после file changes.
```
