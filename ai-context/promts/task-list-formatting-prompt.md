# Промт: Форматирование task-list

## Назначение

Используй этот промт, когда пользователь просит: `отформатируй задачи`.

## Готовый промт

```text
Отформатируй все неотформатированные задачи в `ai-context/tasks/task-list.md`.

Правила:

1. Сохрани порядок задач.
2. Всем неотформатированным строкам добавь код и статус.
3. Используй формат:
🔴 [01-001] Короткое описание задачи
4. Для новых задач используй следующий свободный код в формате `01-XXX`.
5. Для задач без детализации ставь статус `🔴`.
6. Уже отформатированные строки не меняй без необходимости.
7. Если пользователь указал другой сырой источник задач, сначала возьми задачи оттуда и перенеси их в `task-list.md`.

После форматирования:
- обнови `task-list.md`;
- кратко перечисли, какие задачи получили новые коды.
```
