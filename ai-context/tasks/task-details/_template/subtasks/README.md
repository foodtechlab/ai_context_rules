# Subtasks

Эта директория предназначена для детализированных подзадач основной задачи.

Сюда можно добавлять отдельные `.md`-файлы, если:

- в `task-list.md` лежит эпик;
- основную задачу нужно декомпозировать на несколько блоков;
- руководитель проекта или аналитик готовят более детальную структуру для
  разработки.

Рекомендуемый подход:

- один файл = одна подзадача;
- имя файла должно быть читаемым и стабильным;
- в файле подзадачи должна быть явная связь с основной задачей `task-details/<код>/README.md`.
