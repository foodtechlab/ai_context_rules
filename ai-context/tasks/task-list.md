# Task List

Короткий рабочий список задач для AI. Одна строка в секции `Tasks` равна одной
маленькой задаче.

## Рабочий формат

После команды `отформатируй задачи` каждая задача должна выглядеть так:

```text
🔴 [01-001] Короткое описание задачи
```

Где:

- `🔴` - статус задачи.
- `01-001` - код задачи.

## Карта статусов

- `🔴` - новая задача.
- `🟠` - задача детализирована.
- `🟡` - задача в работе.
- `🟣` - AI завершил работу по задаче, но она еще ждет ручной проверки.
- `🟢` - задача вручную проверена пользователем и подтверждена как корректно выполненная.

Правило:

- после завершения своей работы AI переводит задачу в `🟣`;
- в `🟢` задачу переводит только пользователь после собственной проверки.
